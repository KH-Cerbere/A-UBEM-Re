# Conceptual Framework

The review uses a four-layer framework to synthesize microclimate-aware UBEM
methods. This framework is intended to become the core conceptual figure of the
manuscript.

```text
Microclimate data layer
-> Transformation and coupling layer
-> UBEM impact layer
-> Policy-grade reliability layer
```

## Layer 1: Microclimate Data Layer

- Standard EPW / TMY
- Urban weather station
- Fixed sensor network
- Mobile measurement
- Remote sensing
- Reanalysis data
- CFD / ENVI-met / WRF / UWG simulation
- Hybrid observational-simulation data

## Layer 2: Transformation and Coupling Layer

- Weather file replacement
- Weather morphing
- UWG-based transformation
- One-way coupling: UCM -> UBEM
- One-way coupling: UBEM -> UCM
- Two-way coupling / co-simulation
- Surrogate microclimate modeling
- Data assimilation

## Layer 3: UBEM Impact Layer

- Cooling demand
- Heating demand
- Peak load
- Thermal comfort
- Overheating risk
- Retrofit scenario performance
- Carbon emissions
- Grid interaction

## Layer 4: Policy-Grade Reliability Layer

- Validation
- Calibration
- Uncertainty propagation
- Transferability
- Computational scalability
- Data standardization
- Decision usefulness

## Use In Evidence Synthesis

Evidence extraction should connect each paper to this chain only when the
connection is supported by the abstract, PDF note, or extraction table. Do not
infer missing framework layers.
